# Luxury Jewelry Rental Web Application

An enterprise-ready, luxury jewelry rental web application designed with an elegant UI/UX, powered by React 19, Node.js, Express, Prisma ORM, and MySQL.

## Project Structure

- `frontend/`: React 19 + Vite + Tailwind CSS + Framer Motion (Luxury UI)
- `backend/`: Node.js + Express + Prisma + MySQL (MVC REST API)

## Getting Started

1. Set up your MySQL database and copy `backend/.env.example` to `backend/.env` with your connection details.
2. Install dependencies:
   ```bash
   npm install
   npm run install:all
   ```
3. Run the development environment:
   ```bash
   npm run dev
   ```
